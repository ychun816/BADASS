## 🇬🇧

### What is GNS3?

GNS3 is a **network simulation and emulation platform** that allows you to design and test virtual networks on your computer.

It is commonly used by:

* Network engineers
* Cybersecurity students
* DevOps engineers
* Certification candidates (CCNA, CCNP, etc.)

---

### What Can GNS3 Do?

With GNS3, you can:

* Run real router and switch operating systems
* Connect virtual machines
* Integrate Docker containers
* Simulate enterprise-level networks
* Test routing, firewall, VLAN, VPN configurations

It is an **emulator**, meaning it can run real network operating systems (when images are provided), not just simulate behavior.

---

### Main Components

1. **GUI (Graphical Interface)** – Draw and connect devices
2. **GNS3 Server** – Runs and manages devices
3. **Dynamips / QEMU / Docker integration**

   * Dynamips → Cisco router emulation
   * QEMU → Full virtual machines
   * Docker → Lightweight containers

---

### Why It’s Useful for Your Project

If you're building a Docker-based networking project:

* You can create virtual routers using Docker
* Connect multiple containers as hosts
* Test routing between networks
* Build a full virtual lab without physical hardware

It acts like a **virtual networking laboratory**.


## 🇹🇼

### 什麼是 GNS3？

GNS3 是一個**網路模擬與模擬器（Emulator）平台**，可以讓你在電腦上建立並測試虛擬網路環境。

它常被使用於：

* 網路工程師
* 資安學生
* DevOps 工程師
* 網路證照考生（例如 CCNA、CCNP）

---

### GNS3 可以做什麼？

使用 GNS3，你可以：

* 執行真實的路由器與交換器作業系統
* 連接虛擬機器（VM）
* 整合 Docker 容器
* 模擬企業等級網路架構
* 測試路由、Firewall、防火牆、VLAN、VPN 設定

它是「模擬器（Emulator）」，代表它可以真正執行網路設備的作業系統，而不是單純模擬功能。

---

### 主要組成

1. **圖形介面（GUI）** – 用來拖拉並連接網路設備
2. **GNS3 Server** – 負責運行設備與管理模擬環境
3. **Dynamips / QEMU / Docker 整合**

   * Dynamips → Cisco 路由器模擬
   * QEMU → 執行完整虛擬機
   * Docker → 輕量級容器

---

### 為什麼對你的專案有幫助？

如果你正在做 Docker + Networking 專案：

* 可以用 Docker 當成虛擬路由器
* 建立多個虛擬主機
* 測試不同網段之間的 routing
* 不需要實體設備就能建立完整網路實驗室

簡單來說：

Docker 是建立「容器」
GNS3 是建立「整個網路世界」
